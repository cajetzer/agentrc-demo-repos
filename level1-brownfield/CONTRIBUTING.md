# Contributing to ACME API

Thanks for your interest in contributing!

## Getting started

```bash
npm install
npm test        # Run tests
npm run lint    # Check code style
```

## Workflow

1. Create a feature branch from `main`
2. Make your changes with tests
3. Run `npm test` and `npm run lint` — both must pass
4. Open a pull request with a clear description

## Code style

- We use ESLint and Prettier (configs in `.eslintrc.json` and `.prettierrc`)
- Run `npm run lint` before committing
- Write tests for new behavior

## Reporting bugs

Open a GitHub issue with steps to reproduce, expected behavior, and actual behavior.
